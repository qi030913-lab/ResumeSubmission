# Boss Android MVP 实施清单

## 1. MVP 目标

先只做这一条链路：

1. 进入职位详情
2. 点击 `立即沟通`
3. 发送首句
4. 记录结果

不在第一版处理：

- 自动换微信
- 自动多轮聊天
- 自动拨打电话
- 批量海投优化

## 4. 联调顺序建议

建议你按这个顺序打通：

1. ADB 连设备
2. Appium 能拿到当前页面结构
3. 手工定位 `立即沟通`
4. 脚本点击 `立即沟通`
5. 脚本输入首句
6. 脚本点击发送
7. 后端把这套动作包成任务

---

## 5. 完成标准

满足下面 4 条就算 MVP 跑通：

- 能稳定识别 Boss 职位详情页
- 能稳定点击 `立即沟通`
- 能稳定发送模板首句
- 能把结果回传到任务记录里
